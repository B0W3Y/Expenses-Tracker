# Personal Expense Tracker

> 🚧 **Work in progress**
> This project is in early development. This README describes the plan and will be updated as features are built.

## About

This is a full-stack personal expense tracker: an app to log daily expenses, organize them into categories, and visualize spending trends over time. It's being built as a hands-on learning project while studying software engineering, the goal is to practice full-stack development, authentication, database design, and REST API architecture from the ground up.

## Planned Features

- 🔐 User authentication (register / login with JWT)
- 💸 Create, edit, and delete expenses
- 🗂️ Custom categories per user (with default categories on signup)
- 📊 Dashboard with charts (spending by category, monthly trends)
- 🔎 Filter and search expenses by date, category, or amount
- 📥 Import expenses from bank statement CSV files (stretch goal)

## Tech Stack

**Backend**
- Node.js + TypeScript
- Express
- PostgreSQL
- Prisma (ORM)
- JWT + bcrypt for authentication

**Frontend**
- React + Vite
- Tailwind CSS
- Recharts (data visualization)
- TanStack Query
